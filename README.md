# corobus - Message Bus for Coroutines

Реализация шины сообщений для легковесных потоков (корутин) на языке C.

## Описание

Проект представляет собой шину сообщений, в которой можно открывать каналы с ограниченной вместимостью и обмениваться сообщениями между корутинами.

**Ключевые особенности:**
- Блокирующие и неблокирующие операции отправки/получения
- Broadcast (отправка сообщения во все каналы одновременно)
- Автоматическое переиспользование ID каналов при закрытии
- Корректная обработка закрытия каналов с пробуждением спящих корутин
- Отсутствие утечек памяти
